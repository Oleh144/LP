# Responsive Landing Page

This responsive landing page is built using a component-based approach with Vite, React, and Chakra UI.

## Tech Stack

- Vite
- React
- Chakra UI

## Getting Started

1. Install dependencies:
   - `npm install`
2. Start the development server:
   - `npm run dev`
3. Open the app in your browser (typically `http://localhost:5173`).

## Build & Preview

- Build: `npm run build`
- Preview: `npm run preview`

## Deployment

This project can be deployed to GitHub Pages via:

- `npm run deploy`
